You're seeing this error because the Azure Speech Service credentials haven't been configured yet. Follow the steps below to fix this.

1

Get Azure Speech Service Credentials

You need an Azure account with Speech Service enabled. If you don't have one:

  1. Go to Azure Free Account
  2. Create a Speech Service resource in the Azure Portal
  3. Copy your subscription key and region
2

Configure Environment Variables

Edit the .env file in the project root:

# Replace with your actual credentials SPEECH_KEY=your-actual-speech-key-here SPEECH_REGION=eastus # or your region # Optional: For translation API TRANSLATOR_KEY=your-translator-key-here TRANSLATOR_REGION=eastus

Common regions: eastus, westus, westeurope, eastasia

3

Restart the Server

After updating the .env file, restart the server:

# Stop the server (Ctrl+C) # Then restart: npm run dev
4

Test Your Setup

Once configured, test the API endpoint:

curl http://localhost:3001/api/get-speech-token

You should receive a JSON response with a token and region.
